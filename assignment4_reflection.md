# Assignment 4 Reflection

## What worked well
The strongest part of the project was extending the original Gold dataset with Google Trends data and using it to create more meaningful comparison groups.

## What was difficult
The biggest challenge was that Google Trends data was exported weekly, while the Bitcoin dataset was daily. To align them, the weekly trend values were expanded to daily rows.

## Assumptions and limitations
The Google Trends data does not represent exact search counts. It is normalized search interest. Weekly values were expanded across daily rows, which assumes the same interest level across the week. This is a reasonable approximation for this class project, but it may not capture real daily fluctuations.

## What I would improve
If this project were expanded further, I would add more external context such as social sentiment, macroeconomic announcement dates, or other market indicators.
